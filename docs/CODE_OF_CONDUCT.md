# Code of Conduct

Cosmos OSS adopts the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). In plain language:

1. **Be respectful.** Harassment, hate speech, or trolling will not be tolerated.
2. **Assume good intent.** Many contributors are volunteers—work together to find the best solution.
3. **Accept constructive feedback.** Critique code, not people.
4. **Use inclusive language.** Diversity makes the project better.

## Enforcement
- Report unacceptable behavior privately via a GitHub Security Advisory or by emailing the maintainers at `security@cosmos-oss.org` (placeholder inbox—update once the new repo is live).
- The maintainer team will review each report, may request more detail, and will take appropriate action (warning, temporary ban, permanent ban).
- All reports are kept confidential to the extent possible.

## Scope
This policy applies to all project spaces: GitHub issues/PRs, Discussions, community calls, and any official chat channels we may adopt in the future.

By participating in Cosmos OSS you agree to abide by this Code of Conduct.
